# CardBattle

세로형 턴제 카드 배틀 게임 (Android)

## 개요

플레이어와 컴퓨터가 번갈아 턴을 진행하는 1v1 카드 배틀 게임입니다.
8종의 카드로 덱을 구성하고, 전장에서 카드를 배치하여 상대의 모든 카드를 제거하면 승리합니다.

## 서론

*기존 채용 과제의 기획 부분 문제가 있다고 판단하여 게임 진행을 위해 일부 요구사항을 수정하여 개발했습니다.
1) HP만큼 피해를 주는 부분: 현재 HP만큼 피해를 주면 HP 밸런싱이 불가능하며 카드끼리 전투가 성립되지 않습니다. 
각 카드의 개성을 위해 카드의 성장(별레벨)만큼 피해를 주도록 변경하였습니다.
2) 힐러 오버밸린싱: 힐러들이 매턴 모든 카드의 체력을 회복시키면 너무 강력했습니다.
오픈된 카드중 가장 낮은 체력의 카드를 치료하도록 변경했습니다.
3) 스킬 제한: 스킬이 일반 공격보다 좋으므로, 일반 공격을 전략적으로 선택할 이유가 없었기 때문에, 플레이어가 게임당 5회만 스킬을 사용 하도록 변경했습니다.

## 구현 기능 목록

### 타이틀씬
- **상점 패널** - 골드로 카드 구매 (1장: 100골드 / 10장: 900골드), 뒤집힌 카드를 터치하면 카드 플립 연출로 내용 공개
- **도감 패널** - 전체 8종 카드를 그리드로 한 화면에 표시, 각 카드 하단에 보유량/승급 조건(n/10) 표기
- **덱 구성 패널** - 보유 카드 중 6장을 드래그&드롭으로 장착/해제/교체, 6장 미만이면 게임 시작 불가
- **설정 패널** - BGM/SFX 볼륨 슬라이더 조절
- **난이도 선택** - 쉬움/보통/어려움, 선택 즉시 배틀씬 전환
- **종료 확인** - 확인/취소 팝업 후 앱 종료
- **골드 표시** - 타이틀 화면에 현재 보유 골드 실시간 표시

### 배틀씬
- **카드 배치** - 전장 3슬롯 + 대기 3슬롯, 전장 카드 제거 시 대기 카드 자동 승격
- **턴 진행** - 카드 선택 > 행동 선택(공격/스킬) > 대상 선택 > 효과 적용 > 턴 종료 (5단계)
- **카드 타입별 전투 로직**
  - 일반(Normal): 별레벨 기반 공격 + 반격 피해
  - 원거리(Ranged): 반격 피해 없는 원거리 공격
  - 무쌍(Peerless): 대상 + 인접 적 1장에게 추가 피해
  - 힐러(Healer): 턴 시작 시 아군 자동 힐 (패시브)
- **8종 카드 고유 스킬**
  - 전사: 반격 피해 면역 공격
  - 레슬러: 행동불능(스턴) 1턴
  - 궁수: HP 2 이하 즉사
  - 총잡이: 전체 적 공격
  - 기동대: 출혈 2턴
  - 기마병: 스킬 봉인 2턴
  - 의무병: 단일 대상 2 회복 + 상태이상 해제
  - 간호사: 아군 전체 1 회복
- **상태이상 시스템** - 스턴, 출혈, 스킬 봉인 (카드 UI 하단에 남은 턴 수 표시)
- **스킬 횟수 제한** - 플레이어/적 각각 5회
- **적 AI** - 난이도별 분기 로직 (쉬움: 랜덤, 보통: 확률적 스킬 사용, 어려움: 항상 최적 행동)
- **전투 연출**
  - 근접 카드: 대상에게 접근하며 공격 (DOTween)
  - 원거리 카드: 살짝 전진 후 공격
  - 피격 시 흔들림 효과
  - 카드 제거 시 축소+페이드아웃 애니메이션
  - 카드 선택 시 확대 + 테두리 하이라이트
- **데미지 팝업** - 피격/힐 수치 표시, DOTween 팝업 애니메이션 (등장-상승-페이드아웃)
- **즉사 텍스트 팝업** - 궁수 스킬 즉사 시 별도 텍스트 연출
- **HP 표시** - UGUI Slider + 현재HP/최대HP 텍스트
- **배틀 UI** - 턴 표시, 행동 가이드 텍스트, 선택한 카드 정보(이름/타입/스킬 설명), 공격/스킬 버튼
- **승리/패배 결과 패널** - 타이틀 복귀 버튼

### 시스템
- **데이터 저장/로드** - JSON 기반 로컬 저장, 5초 주기 자동 저장 (UniRx Interval), 앱 일시정지/종료 시 저장
- **별레벨 승급** - 같은 카드 10장 수집 시 자동 승급
- **신규 유저 초기화** - 첫 시작 시 전체 카드 1장씩 지급 + 자동 덱 구성
- **사운드 시스템** - SfxDatabase ScriptableObject 기반, BGM/SFX 분리 관리, 볼륨 설정 저장
- **데이터 마이그레이션** - 세이브 파일 무결성 검증 및 복구

│   ├── RuntimeCard.cs      # 런타임 카드 인스턴스 - HP(ReactiveProperty), 상태이상
│   └── SoundManager.cs     # 사운드 재생 관리

### 외부 라이브러리 활용

- **UniRx** - 버튼 바인딩(`OnClickAsObservable`), 카드 HP 변화 감지(`ReactiveProperty`), 자동 저장 타이머(`Observable.Interval`), 배틀 페이즈 상태 관리(`IReadOnlyReactiveProperty`), 이벤트 스트림(`Subject`), 구독 생명주기 관리(`AddTo`)
- **UniTask** - 비동기 턴 진행, 공격/스킬 연출 시퀀싱, 카드 제거 및 슬롯 충원 대기, AI 행동 딜레이
- **DOTween** - 카드 선택 확대/축소, 근접 공격 접근 모션, 원거리 공격 연출, 피격 흔들림(`DOShakePosition`), 카드 제거 애니메이션(`DOScale` + `DOFade`), 카드 플립 연출(`DOScaleX`), 데미지 팝업 등장-상승-페이드아웃 시퀀스
